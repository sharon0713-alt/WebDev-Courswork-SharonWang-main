# Unit 5 Reflection

In Unit 5, I learned how CSS makes web pages look alive. Cascading Style Sheets, or CSS, is a language that lets you style and arrange HTML elements. I learned about basic things like color, font-size (using pixels), font-family (using no more than three fonts for readability), and background-color. I also learned that it is important for readability to have a good contrast between the text and the background. 

Then I learned three ways to use CSS: inline (putting it in the style attribute for quick testing), internal (putting it in the head section with style tags for single pages), and external (putting it in a separate CSS file linked with link tags, which is best for bigger sites because it supports caching and reuse across pages). I also learned that CSS cascades, which means that inline styles usually take precedence. 

Unit 5 taught me how to make my websites look good, be easy to read, and have lots of color.
