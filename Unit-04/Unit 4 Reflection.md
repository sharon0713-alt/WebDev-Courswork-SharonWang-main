# Unit 4 Reflection

I learned how to add attributes, lists, images, and accessibility features to HTML pages in Unit 4.

I learned about HTML attributes like lang and title first. When users hover over elements, the title attribute adds tooltips, and the lang attribute tells screen readers what language the page is in, which helps them say the text correctly.

Then I learned how to make different kinds of lists. Ordered lists (ol) are for steps that are numbered, unordered lists (ul) are for bullet points, description lists (dl with dt and dd) are for words and their meanings, and nested lists let me put lists inside other lists. Lists make it easy to find things.

I also learned how to use the <img> tag to add images. The "src" attribute is for the image source, the "alt" attribute is for alternative text, and the "width" and "height" attributes are for sizing. I learned that absolute links are full URLs and relative links are paths that are relative to the current file. Pictures make web pages more interesting and useful.

I finally learned about how to make websites accessible. Making a website accessible means that more people can read and interact with the content. I learned how to use alt text for images, the right heading hierarchy (h1 to h6), and semantic HTML so that screen readers and text-to-speech software can help people who have trouble seeing or learning.
